Google anuncia el día de hoy por medio de la red social snapchat el nuevo nombre de android N, este será android Nougat que es un turrón de origen Frances hecho con clara de huevo, miel y almendra también existe una variación de la misma que se le pone pistachos.

**La novedades de android Nougat:**

- Multiventanas de forma nativa
- Mejora en el centro de notificaciones
  - Se puede responder directamente del centro de notificaciones
  - Agrupar las notificaciones
- Mejor compatibilidad con idiomas
- Ahorro de datos
  - Puedes bloquear ciertas apps para consumo en segundo plano
- Sincronizar números bloqueados para la entrada
- Reinicio inteligente
  - Permite que si se reinicia el celular de forma inesperado diferentes apps sigan funcionando como la de mensajes, llama y alarmas.
- Android Runtime añade el nuevo compilador Just in Time (JIT)
- Mejora de Doze (Optimiza la bateria)
- Ahora integra la opción de mostrar información de emergencias con el teléfono bloqueado
- Una nueva ventana para reportar errores y reiniciar apps
- API Vulkan para reproducir imágenes en 3D
- Nuevos emojis.
